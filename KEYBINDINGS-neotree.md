# Keybindings Reference

## Claude Code

| Key | Action |
|-----|--------|
| `Ctrl+J` | Insert new line |

## Neovim

### General

| Key | Action |
|-----|--------|
| `Space+e` | Toggle neo-tree |
| `Space+o` | Focus neo-tree |
| `Space+ff` | Find files (Telescope) |
| `Space+fg` | Live grep |
| `Space+fb` | Buffers |
| `Space+fh` | Help tags |
| `Shift+H` | Previous buffer |
| `Shift+L` | Next buffer |
| `Ctrl+H` | Move to left window |
| `Ctrl+J` | Move to lower window |
| `Ctrl+K` | Move to upper window |
| `Ctrl+L` | Move to right window |
| `Ctrl+D` | Scroll down (centered) |
| `Ctrl+U` | Scroll up (centered) |
| `gcc` | Toggle comment |
| `Esc` | Clear search highlight |

### Neo-tree (inside the tree window)

| Key | Action |
|-----|--------|
| `Enter` | Open file / expand folder |
| `l` | Open file / expand folder (custom) |
| `h` | Collapse folder (custom) |
| `a` | Add file |
| `A` | Add directory |
| `d` | Delete |
| `r` | Rename |
| `y` | Copy filename |
| `Y` | Copy relative path |
| `x` | Cut |
| `p` | Paste |
| `c` | Copy |
| `m` | Move |
| `q` | Close neo-tree |
| `R` | Refresh |
| `?` | Show help |
| `<` | Previous source |
| `>` | Next source |

## Zsh

| Key | Action |
|-----|--------|
| `Ctrl+A` | Beginning of line |
| `Ctrl+E` | End of line |
| `Ctrl+R` | Search history |
| `Up arrow` | Search history backward |
| `Down arrow` | Search history forward |
| `Ctrl+Right` | Forward word |
| `Ctrl+Left` | Backward word |
